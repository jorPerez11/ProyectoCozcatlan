# Proyecto Cozcatlan

Cōzcatlan (que en Náhuat significa "Lugar de Joyas") es un emprendimiento salvadoreño diseñado como un puente cultural para conectar a la diáspora y a entusiastas culinarios con la esencia de El Salvador.
